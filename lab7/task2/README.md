# Task 2: Object-Oriented Programming in Python

This project demonstrates the main OOP concepts required for the assignment:

- `Vehicle` is the base class.
- `ElectricCar` and `Motorcycle` are child classes.
- The child classes inherit attributes and methods from the base class.
- `start_engine()` and `vehicle_info()` are overridden to demonstrate polymorphism.
- The code is split into modules with `models.py` and `main.py`.

## Files

- `models.py` - contains the class hierarchy
- `main.py` - creates objects and demonstrates inheritance and polymorphism

## How to run

```bash
python main.py
```

## What the program demonstrates

1. A base class with attributes, methods, and `__str__`
2. Two child classes with unique attributes and methods
3. Inheritance and method overriding
4. Polymorphism through a shared method call on different objects
